    About the System

    This system analyzes ADS-B flight trajectory data to detect potential near-miss events. It uses machine learning combined with trajectory-based features such as distance, altitude separation, and Closest Point of Approach (CPA) to predict risk.

    Unlike traditional monitoring systems, this approach enables predictive safety analysis by identifying conflicts before they occur.
