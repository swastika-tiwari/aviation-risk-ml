Aviation Near-Miss Detection System

About This System

This system analyzes aircraft trajectory data (ADS-B) to identify potential near-miss events. A near-miss occurs when two aircraft come dangerously close in both horizontal and vertical space.

How It Works

  1. Aircraft trajectory data is collected from ADS-B signals.
  2. Aircraft pairs in the same time window are identified.
  3. Distances, altitude differences, and motion patterns are computed.
  4. A machine learning model predicts whether the interaction is risky.

What is a Near-Miss Event?

A near-miss (loss of separation) is defined as:

  • Horizontal distance less than 5 nautical miles (~9.26 km)
  • Vertical separation less than 1000 feet
